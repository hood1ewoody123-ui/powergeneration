# Ассеты PGC

## Структура

| Папка | Назначение |
|-------|------------|
| `_originals/` | Исходники (PNG/MP4/SVG), **не для продакшена** |
| `../public/assets/` | Оптимизированные файлы для сайта |

## Пересборка

```bash
pnpm optimize:assets
```

Требуется: `ffmpeg`, `cwebp` (`brew install webp`), `pngquant` (опционально).

## Что сделано при оптимизации

- **PNG → WebP** с альфой, `q 82–88`, ресайз под экран
- **Видео → H.264** CRF 23–24, `faststart`, без аудиодорожки, лимит разрешения
- **Постеры** — кадр 0.5s → WebP для lazy/facade
- **Латинские имена** файлов (без кириллицы в URL)

`_originals/` в git не коммитится (см. `.gitignore`).
